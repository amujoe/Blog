# canvas 不会就学, 学了还不会, 会了还要学



**首先要了解的是:**  

* canvas  并不非常难! 
* 会一些基本的[HTML](https://developer.mozilla.org/zh-CN/docs/HTML)和[JavaScript](https://developer.mozilla.org/zh-CN/docs/JavaScript)知识, 完全可以快速上手
* 所有的新版本主流浏览器都支持它
* 它可以用于动画、游戏画面、数据可视化、图片编辑以及实时视频处理等方面
* Canvas API主要聚焦于2D图形。
